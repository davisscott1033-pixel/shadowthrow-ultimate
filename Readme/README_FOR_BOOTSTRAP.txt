ShadowThrow Bootstrap Runner
============================

How to use:
1) Extract this ZIP **at the root of your repo** (same folder that has your package.json / pnpm-workspace.yaml).
2) Double‑click **run-shadowthrow-bootstrap.bat**.
3) The script will create/append standard files (docs, CI, .gitignore, .editorconfig, .env.example, etc.).
4) If a Git repo exists, it will try to commit the changes.

If you prefer, you can right‑click **run-shadowthrow-bootstrap.ps1** and choose “Run with PowerShell”.
